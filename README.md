# phytoplankton-zooplankton-GOM

Summary: Phytoplankton and zooplankton form the base of the food web in the ocean. Their interactions with each other helps to kickstart nutrient cycling to higher trophic levels. Understanding who is present as well as who is interacting with who can allow us to better understand the marine food web from a microbial point of view. 

This repository contains my code/analysis about data from a 2023 Gulf of Mexico cruise (TAMU Oceanography, GRADients Cruise).  

Current scripts:
  1. Getting started (making data tables)
  2. RDA Analysis
  3. Various Figures
  4. Beginings of network analysis (involves an HPRC)
